# 📊 Customer Churn Analysis | Data Analytics Project

## 📌 Overview
Customer churn is a critical challenge in many industries. This project focuses on analyzing a telecommunications dataset to identify patterns, trends, and key drivers behind customer churn. The analysis provides actionable insights to improve customer retention strategies.

---

## 🎯 Project Objectives
- Analyze customer behavior and churn patterns
- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Visualize key trends using statistical graphs
- Segment customers based on tenure
- Generate business insights and recommendations

---

## 🛠️ Tech Stack
- Python
- Pandas (Data Processing)
- Matplotlib (Data Visualization)
- Seaborn (Statistical Visualization)

---

## 🔍 Methodology

### 1️⃣ Data Understanding
- Loaded dataset using Pandas
- Explored structure, columns, and data types
- Identified missing values and inconsistencies

### 2️⃣ Data Cleaning
- Removed duplicate entries
- Handled missing values using appropriate techniques
- Standardized column names for consistency

### 3️⃣ Exploratory Data Analysis (EDA)
- Generated summary statistics (mean, median, distribution)
- Created histograms and boxplots
- Analyzed churn vs non-churn distribution

### 4️⃣ Customer Segmentation
- Grouped customers based on tenure:
  - 0–12 months
  - 13–36 months
  - 37+ months
- Visualized customer distribution using pie charts
- Compared average monthly charges using bar charts
